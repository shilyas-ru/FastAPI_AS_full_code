## Уведомления
1. Код учебный, то есть:
    - Содержит все комментарии, которые я пишу сам для себя. С точки зрения стороннего человека комментариев может быть много.
    - Содержит различный код, который писался сугубо для изучения возможностей FastAPI (такой код может иметь "косвенное" отношение к выполняемому проекту, цель кода - отразить различные возможности). К примеру, это м.б. реализация разными способами одной и той же функциональности.
2. Итоговый проект, надеюсь, будет очищен от всех сторонних комментариев и "лишнего", то есть, учебного, кода. Ссылка, надеюсь, будет тут, когда проект доведу до итогового кода.
3. Перед запуском проекта требуется установить пакеты, указанные в файле "[requirements.txt](requirements.txt)", расположенного в соответствующей папке.
4. Начиная с задания "Задание №9: Получение cookie пользователя внутри ручки" требуется создать в корне проекта файл `.env` и заполнить значения, указанные в файле `.env`:
```
DB_HOST=
DB_PORT=
DB_USER=
DB_PASS=
DB_NAME=

JWT_SECRET_KEY=
JWT_ALGORITHM=
ACCESS_TOKEN_EXPIRE_MINUTES=
```
Пример заполнения данных (указывается только по причине того, что проект учебный!!!), так же см. файл [.env-example](.env-example):
```
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASS=postgres
DB_NAME=booking

JWT_SECRET_KEY=09d25e094faa6ca2556c818166b7a9563b93f7099f6f0f4caa6cf63b88e8d3e7
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```
5. Начиная с задания 8 "Задание №8: Запретить создание нескольких юзеров с одинаковой почтой" надо изменить базу данных и создать дополнительные таблицы, как указано в лекциях.
