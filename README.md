## Уведомления
1. Код учебный, то есть:
    - Содержит все комментарии, которые я пишу сам для себя. С точки зрения стороннего человека комментариев может быть много.
    - Содержит различный код, который писался сугубо для изучения возможностей FastAPI (такой код может иметь "косвенное" отношение к выполняемому проекту, цель кода - отразить различные возможности). К примеру, это м.б. реализация разными способами одной и той же функциональности.
2. Итоговый проект, надеюсь, будет очищен от всех сторонних комментариев и "лишнего", то есть, учебного, кода. Ссылка, надеюсь, будет тут, когда проект доведу до итогового кода.
3. Перед запуском проекта требуется установить пакеты, указанные в файле "requirements.txt", расположенного в соответствующей папке.
